# strack-wanted-server

StrackOut x Wanted !

this source is for server-side.



## Project setup
```
npm install
```

### Compiles and minifies for production
```
npm run build
```

### Run server
```
npm run start
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Run typeorm test
```
npm run test-orm
```
